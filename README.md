# School Website

A modern, responsive school website built using React, Vite, and Tailwind CSS.

## Features

- 6 pages — Home, About, Academics, Admissions, Gallery, Contact
- Responsive design — works on mobile, tablet, and desktop
- Masonry photo gallery with category filters and lightbox
- Admissions application form with success state
- Embedded Google Maps on the contact page
- Teal and orange color scheme with Nunito + Inter typography

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM

## Pages

| Page | Description |
|---|---|
| Home | Hero, features, news, and CTA |
| About | Mission, vision, values, and leadership team |
| Academics | Curriculum, subjects, and BECE prep |
| Admissions | Application form and requirements |
| Gallery | Masonry photo grid with category filters |
| Contact | Contact form, Google Maps, and quick links |

## Getting Started
```bash
git clone https://github.com/CJ0819/
cd school0
npm install
npm run dev
```


# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
